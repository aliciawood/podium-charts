# Reporting Table

## Tooltips in Headers Example:
```js
const headers = [
  {
    id: 'name',
    content: 'Name & Phone/Email'
  },
  {
    id: 'sentBy',
    content: 'Sent By'
  },
  {
    id: 'timeSent',
    content: 'Time Sent',
    tooltip: 'I am a tooltip!'
  },
  {
    id: 'followedLink',
    content: 'Followed Link',
    tooltip: 'I am also a tooltip!'
  },
  {
    id: 'review',
    content: 'Review'
  }
];
```
